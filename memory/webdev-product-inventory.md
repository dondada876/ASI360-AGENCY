## 6. KEY GOTCHAS

1. **Flatsome uses UX Builder, NOT Gutenberg** — cannot use Site Factory Engine or transplanting engine. Content is shortcode-based, requiring completely different automation approach.

2. **Porto uses WPBakery shortcodes** — while it supports Gutenberg, the bundled WPBakery is the primary builder. Porto Studio templates use WPBakery format. Transplanting would need a WPBakery-aware engine.

3. **Traveler is a PLATFORM, not just a theme** — at 56MB it includes a complete booking engine. Don't treat it as a simple theme install.

4. **All Envato licenses are Regular License** — covers ONE end product. If used on multiple client sites, each needs its own license.

5. **WC Extra Product Options "Normal mode" is DEPRECATED** — always use Builder mode. Legacy code from pre-builder versions should not be referenced.

6. **Table Rate Shipping rows are priority-ordered** — LAST qualifying row wins, not first. This is counter-intuitive.

7. **Ultimate Affiliate Pro has version conflicts** — the plugin explicitly handles deactivation of `indeed-affiliate-pro` (legacy version) and `ultimate-affiliate-lite` versions. Only one version should be active.
